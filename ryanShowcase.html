Aviation Dependency Analyzer

This tool is meant to be used as a tool to analyze code dependencies and targets as understood by the boost-build system. The two tabs at the top of the window provide two different views of the projects dependencies.


Dependency Tool

  • Select a node to view summary information
  • Select the 'Show ACL Warnings' button to display bad architecture
  • Use the 'Search Nodes' text box to jump to specific nodes

Edge Bundle

  • Use the 'Search Nodes' text box to quickly cycle through nodes
  • Click a node to select and deselect it and view its summary information
  • Use the slider at the bottom-left of your screen to change the link display
